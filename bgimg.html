background 배경

  1. background-color:색상
  2. background-image:url("배경이미지경로");
  • background-repeat: ; 배경이미지 반복 지정
    1. repeat (기본) 양사방 반복
    2. repeat-x 가로로만 반복
    3. repeat-y 세로로만 반복
    4. no-repeat 한번만 출력(반복X)
  • background-position: ;배경이미지 위치 지정
    1. left/center/right/top/center/bottom 방향값을 조합(수평 수직 순)
    2. 원하는 값을 지정 (Xn Yn)
  • background-size: ; 배경이미지 크기 지정
    1. cover : 이미지 하나로 영역 전체를 덮음(이미지가 잘려나갈수도)
    2. contain : 이미지 하나로 영역 전체를 덮지만 이미지는 온전히 다 보임
    3. 원하는 값을 지정 (Xn Yn)
  • background-attachment: ; 배경이미지 고정여부 지정
    1. scroll(기본) 스크롤바를 따라 움직임
    2. fixed 고정
    3. local

배경이미지 여러개 넣기 주소, 속성에 쉼표로 구분하여 설정해준다 { font-size:1.2em; line-height: 1.8; background-image: url("img/c3.jpg"), url("img/c5.png"); background-repeat: no-repeat, repeat-y; background-position: 0 center, 100% center; background-attachment: fixed; }

background-attachment: ; 배경이미지 고정여부

scroll - 자신의 영역에는 고정, 브라우저에는 스크롤
fixed - 모두 고정
local - 모두 스크롤
